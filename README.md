# DOKO*2(どこどこ)であなたの旅をもっと快適に!

技育CAMPハッカソン 2025年度 Vol.17にてチームJAZZSが開発したお出かけ支援アプリDOKO*2(どこどこ)のリポジトリです。

## プロジェクト概要

旅行先で周辺のおすすめスポットを検索する際にsnsやmapアプリなど複数のアプリをまたぐことが煩わしい、お出かけ先で少し時間ができたから時間をつぶしたいけどちょうどよい場所を探すのが難しい等、出先で今の気分に合わせた場所を探すのは意外と難しい、という課題からこのアプリの開発が始まりました。現状として、マップアプリは行き先が決まっている場合のルート検索がメインですが、DOKO*2はその時の気分に合わせた場所を提示してくれる旅行支援的側面が大きいマップアプリとなっています。

## 機能

1. おすすめスポット検索機能

現在地から周辺の情報を取得し、好きな個数の観光地をマップ上に表示してくれます。

2. AIチャット機能

チャット欄にチャットを送ることで今の状況に最適な場所の情報を返信として、その場所を地図上にマーカーとして表示してくれます。
